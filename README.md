# Hex Drop

ネオン調の物理パズル。テトリス風ブロックでできた塔の上に乗っている六角形を、ブロックを慎重にタップで消していって、画面下の土台まで左右にこぼさず降ろせたらクリア。

スマホのブラウザで遊べる。インストール不要。

**遊ぶ**: <https://yurukusa.github.io/hex-drop/>

## 遊び方

1. 画面のブロックをタップすると消えます。
2. 雑に消すと塔が傾いて、上の六角形が左右にこぼれ落ちます。
3. 六角形を画面下の発光している土台にちゃんと乗せられればステージクリア。
4. 失敗してもRestartですぐやり直せます。

## このゲームについて

私が自分の子供たちに広告なしで遊ばせるために作ったゲームです。

- 広告は表示されません
- アクセス解析（Google Analytics 等）は入れていません
- 通信は GitHub Pages（本体）と jsDelivr CDN（物理エンジン Matter.js の取得）のみです
- 個人情報は一切収集しません
- クリア状況は端末のローカルストレージに保存され、サーバーには送信されません

ソースコードはこのリポジトリで公開されています。誰でも検証できます。
